Frequently Asked Questions

Everything you need to know about your audio palette

What makes PadVibe different from other audio palette apps?

PadVibe stands out with its zero-lag playback engine, cross-platform compatibility, and intuitive drag-and-drop interface. Unlike competitors, PadVibe was built from the ground up for professional audio performance while maintaining an easy-to-use experience.

Is PadVibe free to use?

Yes! PadVibe is currently available as a free download for all users. We may introduce premium features in the future, but our core audio palette functionality will always remain free.

What audio formats does PadVibe support?

PadVibe supports all common audio formats including MP3, WAV, FLAC, OGG, and AIFF. This ensures compatibility with virtually any audio file you might want to use in your audio palette.

Can I use PadVibe with my streaming software?

Absolutely! PadVibe works seamlessly with OBS, Streamlabs, XSplit, and other popular streaming software. It can be configured to output audio to any virtual or physical audio device on your system.

How many sound pads can I use in my audio palette?

PadVibe supports up to 64 audio pads in a single audio palette. You can also create multiple pad configurations (called "decks") and switch between them instantly during your sessions.

Is there a web version of PadVibe?

Yes! PadVibe is available as a web application that runs directly in your browser. While the desktop versions offer some additional features, the web version provides the same core audio palette experience without any installation required.
